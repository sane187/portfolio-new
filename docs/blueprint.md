# **App Name**: Arpit Joshi Portfolio

## Core Features:

- Project Gallery: Display professional projects with descriptions, images, and links. The LLM tool analyzes project details to automatically categorize and tag projects.
- Skill Visualization: Interactive display of skills, experiences and tech stack from the provided CV using charts and graphs.
- Responsive Layout: Ensure the portfolio is fully responsive and optimized for every device, from desktop to mobile.
- About Me Section: Highlight skills, experience, and personal details. Dynamically generate sections from the provided CV to display employment history, education, and other details.
- CV parsing: Parse the user provided CV for education, job history and other details to display in the portfolio.

## Style Guidelines:

- Primary color: Teal (#008080) for a touch of elegance and modernity.
- Background color: Dark gray (#222222) to provide contrast and sophistication.
- Accent color: Light teal (#B0E0E6) for highlights and interactive elements. It offers a lighter contrast against the dark gray, guiding user attention without overwhelming the primary teal.
- Font: 'Inter' (sans-serif) for a clean, modern look. Suitable for both headlines and body text.
- Use a grid-based layout for consistent spacing and alignment across all sections.
- Employ a set of simple, line-based icons to represent skills and technologies.
- Subtle transition animations to enhance user experience and create a smooth flow between sections.
